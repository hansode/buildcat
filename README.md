Buildcat
========

[Jenkins](http://jenkins-ci.org/http://jenkins-ci.org/) based build cluster building tools with [vmbuilder](https://github.com/hansode/vmbuilder).

setup
-----

```bash
$ make
```

build targets
-------------

+ Cluster-host
+ Cluster-node
